# Udemy Clone Website 🎓

This project is a simple clone of the Udemy website built using HTML5 and CSS3.  
It is designed to practice frontend web development skills by recreating the layout and design of an online learning platform.

## 🚀 Features

- Responsive Navigation Bar
- Course Categories Section
- Featured Courses Display
- Clean User Interface
- Responsive Design for Different Screen Sizes
- Footer Section

## 🛠️ Technologies Used

- HTML5
- CSS3
